素材管理系统需求报告
-用户类型 
	-学生、管理员
-关系与属性
	-用户(user)
		-用户ID(id) int
		-账号 char(20)
		-姓名(name) char(20)
		-密码(password) char(20)
		-学号(student_number) int
		-权限(privilege) in（'学生'(S) or '管理员'(A)） char(5)
	-素材(material)
		-素材ID(id) int
		-素材名称(name) char(50)
		-上传者ID(uploader_id) int
		-拍摄日期(time) datetime
		-备注(comments) char(300)
	-标签(tag)
		-标签ID(id) int
		-标签名称(name) char(20)
	-素材标签(material_tag)
		-素材ID(material_id) int
		-标签ID(tag_id) int
	-完成人员(material_user)
		-素材ID(material_id) int
		-用户ID(user_id) int
-用户权限
	-学生
		-登录与退出（验证账号、密码）
		-上传素材
		-删除自己上传的素材
		-查看所有素材
	-管理员
		-登录与退出（验证账号、密码）
		-上传素材
		-删除素材
		-查看所有素材
-用户视图
	-登录界面（账号栏、密码栏、登录按键）
	-用户
		-个人姓名、学号
		-退出按键
		-上传素材按键
		-素材列表
			-素材名称(点击显示详细信息)、素材标签、上传日期、删除按键(如果是此用户上传的)
	-管理员
		-个人姓名
		-退出按键
		-上传素材按键
		-素材列表
			-素材名称(点击显示详细信息)、素材标签、上传日期、删除按键
	-上传界面
		-输入框（素材名称、素材标签、拍摄日期、备注、完成人员）
		-上传按键
	-素材详细信息界面
		-素材内容（上传者、素材名称、素材标签、拍摄日期、备注、完成人员）
		-返回列表按键
